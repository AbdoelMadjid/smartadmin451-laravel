## Installer
```console
git clone https://github.com/AbdoelMadjid/smartadmin451-laravel.git
```

```console
composer install
```

```console
composer dump-autoload
```

```console
cp .env.example .env
```

```console
php artisan key:generate
```

```html
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=smartadmin451-laravel
DB_USERNAME=root
DB_PASSWORD=
```

```console
php artisan migrate
```
```console
php artisan db:seed
```
